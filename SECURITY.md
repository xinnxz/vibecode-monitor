# Security Policy

## Supported Versions

Currently, only the latest `main` branch version of Vibe Code Monitor is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v2.x.x  | :white_check_mark: |
| v1.x.x  | :x:                |

## Reporting a Vulnerability

We take the security of Vibe Code Monitor seriously. If you discover a security vulnerability, we'd like to know about it so we can take steps to address it as quickly as possible.

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them privately to the project maintainers via email or direct message. You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible.
